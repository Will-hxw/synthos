# 02 - nt_msg.db · 核心聊天数据库

> `nt_msg.db` 是 QQ NT 体积最大、最重要的数据库文件，包含群聊和私聊的聊天消息数据。

---

## 2. `group_msg_table` — 群聊消息

群聊消息的核心表，存储每条群聊消息的元数据和内容。

### 2.1 列定义

| 列名 | 类型 | MsgRecord 字段 | 说明 |
|:---|:---|:---|:---|
| `40001` | int | `msgId` | 消息 ID，具有唯一性 |
| `40002` | int | `msgRandom` | 消息随机值，用于消息去重 |
| `40003` | int | `msgSeq` | 群聊消息序号，在每个聊天中依次递增 |
| `40005` | int | — | 作用不明，仅自己发的消息有一定概率存在数值，正常为 0 |
| `40006` | int | — | elem id（作用不明） |
| `40010` | int | `chatType` | 聊天类型（见 [第 3 节](#3-枚举值参考)） |
| `40011` | int | `msgType` | 消息类型（见 [第 3 节](#3-枚举值参考)） |
| `40012` | int | `subMsgType` | PB 子消息类型（见 [第 3 节](#3-枚举值参考)） |
| `40013` | int | `sendType` | 发送标志（见 [第 3 节](#3-枚举值参考)） |
| `40020` | str | `senderuid` | `nt_uid`，对应 `nt_uid_mapping_table` |
| `40021` | str | `peeruid` | 会话 ID（群号 nt_uid 形式） |
| `40027` | int | `peeruin` | 会话 ID（群号 QQ 号形式） |
| `40030` | int | — | 群号（QQNT 保存） |
| `40033` | int | — | 发送者 QQ 号（QQNT 保存） |
| `40041` | int | `sendStatus` | 发送状态（见 [第 3 节](#3-枚举值参考)） |
| `40050` | int | `msgTime` | 消息时间戳（UTC+8，单位：秒） |
| `40058` | int | — | 当日 0 时整的时间戳（GMT+0800） |
| `40060` | int | — | 已退出/已解散群聊标志 |
| `40062` | protobuf | — | 表态信息详情（含表态表情和表态数量），数字与 [QQBot 表情编号](https://bot.q.qq.com/wiki/develop/api/openapi/emoji/model.html) 对应（超级表情不在此列） |
| `40080` | protobuf | — | **消息内容 Protobuf**（⚠️ 早期文档记为 `40080`，更新文档修改为 `40800`；实际多个独立验证源确认为 `40800`） |
| `40083` | int | — | 表态表情数量总和 |
| `40084` | int | — | 表态表情数量总和 |
| `40090` | str | `sendMemberName` | 发送者群名片（旧版格式 `name(12345)`，QQNT 中为群名片） |
| `40093` | str | `sendNickName` | 发送者昵称（旧版 QQ 为空，QQNT 中未设群名片时才有） |
| `40100` | int | — | @ 状态（见 [第 3 节](#3-枚举值参考)） |
| `40600` | protobuf | — | 状态标志（见 [第 3 节](#3-枚举值参考)） |
| `40801` | protobuf | — | 作用不明的 protobuf 数据 |
| `40850` | int | — | 回复消息序号（该消息所回复消息的 msgSeq） |
| `40900` | protobuf | — | 按 msgType 区分：`8` → 转发聊天缓存，`9` → 引用消息 |

> **⚠️ 字段 40800 vs 40080**：社区早期分析（linux.do 原帖）将群聊消息内容记为 `40080`，但 QQDecrypt 官方文档和多个独立实现（star-picker、NapNeko 等）均使用 `40800`。本项目 proto 定义经实际数据验证采用 `40800`。

### 2.2 `group_at_me_msg` — 被 @ 消息

| 列名 | MsgRecord 字段 | 含义 |

|:---|:---|

| `41103` | 置顶时间 |
| `1000` | 私聊置顶 → `uin` |
| `60001` | 群聊置顶 → `peeruin` |

| 列名 | MsgRecord 字段 | 含义 |

| `40030` | — | C2C 才有 `uin`，群聊此值为 0 |

| `40094` | — | 来源 |
| `40093` | `sendNickName` | 昵称 |
| `40090` | `sendMemberName` | 群名片 |
| `40095` | `sendRemarkName` | 备注名称 |
| `40020` | — | `nt_uid` |
| `40033` | — | `uin` |

## 3. 枚举值参考

### 3.1 `chatType` 枚举（列 `40010`）

| 值 | 含义 |
|:---|:---|
| `1` | 私聊（C2C） |
| `2` | 群聊 |
| `4` | 频道 |
| `100` | 临时会话 |
| `102` | 企业客服 |
| `103` | 公众号 |

### 3.2 `msgType` 枚举（列 `40011`）

| 值 | 含义 |
|:---|:---|
| `0` | 无消息（消息损坏，多见于已退出群聊且时间久远） |
| `1` | 消息空白（msgId 存在，内容未加载） |
| `2` | 文本消息 |
| `3` | 群文件 |
| `4` | 未发现（待补充） |
| `5` | 系统（灰字）消息 |
| `6` | 语音消息 |
| `7` | 视频文件 |
| `8` | 合并转发消息 |
| `9` | 回复类型消息 |
| `10` | 红包 |
| `11` | 应用消息 |
| `17` | 表情包/原创表情 |

| `8` | 原创表情包 | msgType=17 |

### 3.4 `sendType` 枚举（列 `40013`）

| 值 | 含义 |
|:---|:---|
| `0` | 他人发送 |
| `1` | 本机发送 |
| `2` | 其他客户端发送 |
| `5` | 转发消息 |
| 其他 | 在已退出/封禁消息中为当日整点时间戳 |

### 3.5 `sendStatus` 枚举（列 `40041`）

| 值 | 含义 |
|:---|:---|
| `0` | 发送被阻止（如非好友） |
| `1` | 尚未发送成功（如网络问题） |
| `2` | 发送成功 |
| `3` | 消息被和谐 |

### 3.6 @ 状态枚举（列 `40100`）

| 值 | 含义 |
|:---|:---|
| `0` | 不包含 @ |
| `2` | 有人 @ 他人 |
| `6` | 有人 @ 我 |

### 3.7 `40600` 状态标志解析

| 十六进制值 | 含义 | 备注 |
|:---|:---|:---|
| `14 00` | 回复消息 | 此时 `40100`：`6` = 有人回复自己，`2` = 他人回复他人 |
| `c2e91304a8d114**` | 撤回消息 | 值不唯一 |
| `c2e60900` | 好友标志 | 仅在 `profile_info_v6.20072` 中出现 |

## 4. `c2c_msg_table` — 私聊消息

| 列名 | 类型 | 含义 | 说明 |
