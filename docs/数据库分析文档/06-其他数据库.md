## 1. emoji.db — 表情包数据库

### 1.1 表一览

| 表名 | 说明 |
|:---|:---|
| `base_sys_emoji_table` | QQ 默认系统表情数据 |
| `bottom_emoji_table` | 账号下已收藏的原创表情数据（protobuf 格式） |
| `emoji_config_storage_table` | QQ 系统表情配置数据 |
| `emoji_group_table` | 表情分组说明（如超级表情 0-16） |
| `emoji_misc_data_table` | QQ 默认表情下载地址 |
| `fav_emoji_info_storage_table` | 用户收藏的自定义表情 |
| `market_emoticon_package_table` | 原创表情市场（表情包列表） |
| `market_emoticon_table` | 已下载的原创表情信息 |

### 1.2 `base_sys_emoji_table` — 系统表情

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `81211` | str | 表情 ID | — |
| `81212` | str | 表情描述 | 外显文字（如 `[微笑]`） |
| `81218` | protobuf | 下载链接 | 包含 `81229` 和 `81230` 的链接内容 |
| `81221` | int | 特殊类表情标识 | `0` = 正常，`1` = 特殊 |
| `81224` | int | 特殊限定类表情标识 | — |
| `81225` | int | 特殊限定类表情标识 | — |
| `81226` | int | EmojiType | `1` = 系统表情，`2` = emoji 表情，`3` = 动态可变表情（如骰子） |
| `81266` | int | 表情类型说明 | — |
| `81229` | str | 静态图片下载地址 | CDN URL |
| `81230` | str | APNG 图片下载地址 | emoji 表情无此链接 |

### 1.3 `fav_emoji_info_storage_table` — 收藏表情

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `80001` | int | 表情排序 | 倒序排列：`0` = 最新收藏 |
| `80002` | str | 文件名 | 格式：`{uin}_0_0_0_{MD5}_0_0` |
| `1002` | str | 用户 uin | 收藏者 QQ 号 |
| `80010` | str | 下载地址 | 格式：`https://p.qpic.cn/qq_expression/{1002}/{8002}/0` |
| `80011` | str | 表情 MD5 | 32 位大写 |
| `80012` | str | 本地存储路径 | 缓存路径 |
| `80201` | str | 原创表情标识符 | `80213`=1 时有效 |
| `80202` | int | 原创表情包 ID | `80213`=1 时有效 |
| `80213` | int | 是否为原创表情 | `0` = 否，`1` = 是 |
| `80223` | str | 表情备注 | — |
| `80225` | str | 表情备注 | — |

### 1.4 `market_emoticon_package_table` — 原创表情市场

| 列名 | 含义 | 说明 |
|:---|:---|:---|
| `80943` | 原创表情包 ID | 已知 ID 可在 `../{uin}/nt_qq/nt_data/Emoji/marketface/{ID}` 找到缓存的原创表情包 |
| `80947` | 原创表情包名字 | 表情包名称 |
| `80948` | 表情包说明 | 表情包描述 |

### 1.5 `market_emoticon_table` — 已下载原创表情

| 列名 | 含义 | 说明 |
|:---|:---|:---|
| `80920` | 表情标识符 | — |
| `80943` | 原创表情包 ID | 关联 `market_emoticon_package_table` |
| `80921` | 表情描述 | 外显文字 |

## 2. collection.db — 收藏数据库

### 2.1 表一览

| 表名 | 说明 |
|:---|:---|
| `collection_list_info_table` | 收藏基本信息（来源、摘要等） |
| `content_list_info_table` | 收藏实际内容（仅聊天记录和笔记类收藏有此表） |

### 2.2 `collection_list_info_table` — 收藏列表

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `180001` | str | `sid` | 收藏唯一标识符 |
| `180008` | int | `type` | 收藏类型（见下表） |
| `180009` | int | 创建/收藏时间 | 秒级时间戳 |
| `180011` | int | 最后编辑时间 | 秒级时间戳 |
| `180004` | protobuf | `Source Info` | 来源信息（见下表） |
| `180015` | protobuf | `Summary Info` | 摘要信息（见下表） |

#### 收藏类型（`180008`）

| 值 | 说明 |
|:---|:---|
| `1` | 来源于聊天记录（文本类消息） |
| `2` | QQ 笔记 |
| `9` | 带有链接的聊天消息 |

#### 来源信息（`180004` Protobuf）

| Field Number | 说明 |
|:---|:---|
| `18504` | 收藏来源群号 |
| `18505` | 群名称 |
| `18506` | 发送者 `nt_uid` |
| `18501` | 发送者 QQ 号 |
| `180503` | 备注名 / 群员名称 / 用户昵称 |

#### 摘要信息（`180015` Protobuf）

| Field Number | 说明 |
|:---|:---|
| `181450` | 标题（可为空） |
| `181452` | 外显描述内容 |
| `181453` | 图片信息 |
| `18504` | 来源会话 |
| `180550` | 图片 URL |
| `180553` | 图片名称 |
| `180561` | 图片本地缓存路径 |
| `180610` | 文件保存位置 |

### 2.3 `content_list_info_table` — 收藏实际内容

> 仅聊天记录和笔记类收藏存在此信息。

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `180700` | str | `sid` | 与 `180001` 对应，用于关联收藏 |
| `180720` | int | 最后编辑时间 | 与 `180011` 对应 |
| `180708` | protobuf | 收藏实际内容 | 收藏的消息 protobuf 数据 |

## 3. files_in_chat.db — 聊天媒体文件

存储下载和接收的图片、语音、视频、文件等媒体信息。

### 3.1 `files_in_chat_table`

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `40001` | int | `msgId` | 消息 ID |
| `45001` | int | `clientSeq` | 接收的图片顺序，在本地数据库中递增 |
| `82300` | int | `msgRandom` | 消息随机值，用于消息去重 |
| `45403` | str | `filePath` | 文件存储路径 |
| `45404` | str | `thumbPath` | 预览图缓存路径（多见于视频） |
| `40020` | str | `nt_uid` | 发送者 `nt_uid` |
| `40021` | str | `peerUid` / `senderUin` | 来源群号 / QQ 号 |
| `40010` | int | `chatType` | 聊天类型 |
| `82301` | int | — | 未查明 |
| `45002` | int | `elementType` | 元素类型 |
| `45003` | int | `subElementType` | 子元素类型 |
| `45402` | str | `fileName` | 文件名 |
| `45405` | int | `fileSize` | 文件大小（KB） |
| `40050` | int | `msgTime` | 发送时间 |
| `82302` | int | `original` | 是否为原图：`1` = 是，`0` = 否 |

> ⚠️ 此表主要存储下载文件、语音文件、视频文件的信息。大部分图片的 `45403` 值为空（图片可能通过 CDN 实时下载而非本地缓存）。

## 6. 其他数据库

### 6.2 guild_msg.db — 频道消息

### 6.4 gpro_v1-6_{nt_uid}.db

### 6.5 全文搜索索引（FTS）系列

| 文件 | 说明 |
|:---|:---|
| `group_msg_fts.db` | 群聊消息全文搜索索引 |
| `buddy_msg_fts.db` | 私聊消息全文搜索索引 |
| `discuss_msg_fts.db` | 讨论组消息全文搜索索引 |
| `data_line_msg_fts.db` | 数据线消息全文搜索索引 |
| `msg_fts.db` | 综合消息全文搜索索引 |

这些文件使用 SQLite FTS5 扩展，由 QQ 内部自动维护，通过自定义分词器建立。通常无需手动解析，对应的影子表包含 `_fts`、`_fts_config`、`_fts_data`、`_fts_docsize`、`_fts_idx` 后缀。
