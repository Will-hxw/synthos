# 05 - group_info.db · 群聊信息

> 存储群聊相关的信息：群成员、群公告、群通知、精华消息、群详情等。

---

## 1. 数据库表一览

| 完成解析 | 表名 | 说明 |
|:---:|:---|:---|
| ✅ | `group_notify_list` | 群通知（加群申请、管理员变更等） |
| ✅ | `doubt_group_notify_list` | 过滤群通知（与群通知结构类似，存放被过滤的消息） |
| ✅ | `group_bulletin` | 群公告（仅存最新一条） |
| ✅ | `group_detail_info_ver1` | 群聊详细信息 |
| ✅ | `group_list` | 群聊名称列表（信息与 `group_detail_info_ver1` 对应） |
| ✅ | `group_member3` | 群成员信息 |
| ✅ | `group_member_level_info` | 群等级头衔信息 |
| ✅ | `group_essence` | 群精华消息 |
| 🔵 | `group_avatar` | 群头像（据名称推断，未经验证） |

---

## 2. `group_notify_list` / `doubt_group_notify_list` — 群通知

两个表结构相同，`doubt_group_notify_list` 存放被过滤/风险标记的通知。

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `61001` | int | 消息时间戳 | 删除末三位后为毫秒级时间戳 |
| `61002` | int | 通知类型 | 见下表 |
| `61003` | int | 验证状态 | `0` = 正常，`1` = 过滤，`2` = 同意，`3` = 拒绝，`4` = 忽略 |
| `61004` | protobuf | 群组信息 | 群号与群名称 |
| `61005` | protobuf | 被操作者 | `nt_uid` 与 QQ 昵称 |
| `61006` | protobuf | 操作者 | 仅 `group_notify_list`，`nt_uid` 与 QQ 昵称 |
| `61007` | protobuf | 操作人信息 | 仅 `doubt_group_notify_list`，`nt_uid` 与 QQ 昵称 |
| `61008` | int | 操作时间戳 | 秒级时间戳 |
| `61009` | — | — | 待分析 |
| `61010` | str | 对方附加说明 | 如申请加群时的验证消息 |
| `61011` | str | 系统附加说明 | 如风险提示 |

### 2.1 `61002` 通知类型枚举

| 值 | 含义 |
|:---|:---|
| `1` | 申请加群 |
| `3` | 被设置为管理员 |
| `6` | 被移出群聊 |
| `11` | 被管理员拒绝加入 |
| `13` | 自主退出群聊 |
| `15` | 被取消管理员权限 |

---

## 3. `group_bulletin` — 群公告

仅存储最新一条群公告。

| 列名 | 类型 | 含义 | 说明 |
|:---|:---|:---|:---|
| `60001` | int | peerUid | 群号 |
| `64205` | protobuf | 群公告内容 | protobuf 格式 |

### 3.1 `64205` Protobuf 结构

| Byte Range | Field Number | Type | 含义 |
|:---|:---|:---|:---|
| 0-1089 | `64205` | protobuf | 外层容器 |
| ├─ 0-8 | `60001` | varint | 群号 |
| └─ 8-1084 | `64202` | protobuf | 公告详情 |
| &nbsp;&nbsp;&nbsp;├─ 0-8 | `60001` | varint | 群号（冗余） |
| &nbsp;&nbsp;&nbsp;├─ 8-36 | `64221` | string | 发布公告者 `nt_uid` |
| &nbsp;&nbsp;&nbsp;├─ 40-76 | `64223` | string | fid |
| &nbsp;&nbsp;&nbsp;├─ 80-88 | `64225` | varint | `msgTime`（消息发布时间） |
| &nbsp;&nbsp;&nbsp;├─ 88-96 | `64226` | varint | `ctime`（公告创建时间） |
| &nbsp;&nbsp;&nbsp;└─ 4-17 | `64452` | string | 群公告内容文本 |

## 8. 数据关联关系

```
group_member3.60001 (群号)
    └── group_detail_info_ver1.60001 (群号)
        ├── group_detail_info_ver1.60007 (群名称)
        └── group_detail_info_ver1.60002 (群主 nt_uid)

group_member3.1000 (nt_uid)
    └── profile_info_v6.1000 (nt_uid)
        ├── profile_info_v6.20002 (昵称)
        └── profile_info_v6.1002 (uin)

group_essence.(67501, 67502) (msgSeq, msgRandom)
    └── group_msg_table.(40003, 40002) (msgSeq, msgRandom)
```
